# Reconciliation Eval Datasets

One example per scenario, generated by `python evals/build_evals.py`. Each row uses a different combination of record count, match distribution, 1:N aggregation, and variance to exercise a spread of difficulty levels. Every scenario declares a single monetary key plus a configurable number of non-monetary mapping keys (1-3, see the **# Keys** column), mirroring the get_table_keys_suggestion distribution.
